# About page

This page tells you a little bit about me.

I am a mechatronics engineer and started this project back in 2017 as a hobby project for myself. I like to play around with electronics and some coding feels also relaxing.
The first prototype was put together from discrete modules running the opensprinkler firmware in spring of 2018.
One year later i have ordered the pcb-s matching the original opensprinkler DC variant. The first pcb-s were tested by some friends and myself. In some cases the inter pcb connectors started to oxidize causing hard to debug issues.
This is wy i started to design a single pcb variant of opensprinkler. In 2021 the first self designed pcb boards arrived and dived into deep testing. Since i have made several versions to match standard electrical junction boxes that provide IP55 protection to the controller board.
